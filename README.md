# csci5409-hire

CSCI5409 Cloud computing project for the part-time job portal.

###Docker instructions:  
Create docker image:  
```sudo docker image build -t hiree:1.0 .```  
List all images:   
```sudo docker image ls```  
Run the docker container:  
```sudo docker container run -p 7000:7000 -it hiree:1.0```
